@@ meta
name = live_coding.backup()

@@ signatures
live_coding_backup(backup_name: str = None) -> None

@@ variables
backup_name: str = None - name of file for screenshot and code archive

@@ description
Create a screenshot of the current Sketch window and a copy of the current code. This function combines the functionality of [](py5tools_live_coding_screenshot) and [](py5tools_live_coding_archive_code). Everything will be saved to the archive directory. By default, this is an `archive` subdirectory under the Sketch code's current working directory.

See the documentation for [](py5tools_live_coding_screenshot) and [](py5tools_live_coding_archive_code) for more information about each feature.

Look at the online ["Live Coding"](/content/live_coding) documentation to learn more.

@@ example
import py5_tools

update_count = py5_tools.live_coding.get_update_count()
py5_tools.live_coding.backup(f"version_{update_count}")
